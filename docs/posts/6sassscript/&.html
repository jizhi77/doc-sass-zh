&符

父级选择器引用

.foo.bar .baz.bang,
.bip.qux {
    &.name {
        font: 12px;
    }
    ;
}

// 编译输出
.foo.bar .baz.bang.name,
.bip.qux.name {
  font: 12px;
}

如果没有父选择器，则＆的值将为null。这意味着您可以在mixin中使用它来检测父选择器是否存在：

@mixin does-parent-exist {
  @if & {
    &:hover {
      color: red;
    }
  } @else {
    a {
      color: red;
    }
  }
}

@include does-parent-exist();
div {
  @include does-parent-exist();
}

// 输出CSS
a {
      color: red;
}
div:hover {
  color: red;
}
